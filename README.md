# Excel_Crowdfunding_Workbook

## This workbook demonstrates use of advanced Excel techniques and data visualization skills. 
<img width="1440" alt="Excel Crowdfunding Main" src="https://github.com/user-attachments/assets/f0e85bfb-ebbd-412b-842e-7549f321cf95">

### A crowdfunding dataset was used containing 1000 crowdfunding projects with 18 unique data points for each project.
This project contains 7 separate pages breaking the dataset down by unique datapoints to derive actionable insights from.
Pivot tables were used to be able to see slices of the data on demand based on the available data points.

<img width="1435" alt="Excel Crowdfunding Pivot Table with Visualization" src="https://github.com/user-attachments/assets/43eb47ed-2d05-4ce1-a336-0eb8008fdd75">
Visualizations have been added to easily digest data.

<img width="1224" alt="Excel Crowdfunding Statistical analysis" src="https://github.com/user-attachments/assets/18324c32-3ce4-4492-adbe-ec1dbdf9a622">
Pivot tables are used to be able to visualize different data points. Data has also been conditionally highlighted to easily identify trends in the data.
